## 📌 Project Overview

This is a responsive personal portfolio website developed as **Task 1** of the **Future Interns Full Stack Web Development Internship**.

The website showcases my skills, education, projects, internship work, and contact details in a clean and professional format. It is built using HTML, CSS, and JavaScript and deployed using GitHub Pages.

---

## ✨ Key Features

- Responsive design for laptop and Android/mobile screens
- Sticky navigation menu
- Hero section with animated particle background
- About section with skills and education
- Projects section with project cards
- Downloadable project reports
- Contact form connected with Formspree
- Email, LinkedIn, and GitHub links
- Smooth scrolling and reveal animations
- Hamburger menu for mobile navigation

---

## 🛠️ Tech Stack

| Technology | Usage |
|-----------|-------|
| HTML5 | Website structure |
| CSS3 | Styling, layout, responsiveness |
| JavaScript | Animations, form interaction, hamburger menu |
| Formspree | Contact form message handling |
